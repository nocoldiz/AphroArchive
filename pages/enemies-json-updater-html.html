RPG Maker MZ Enemies.json Battler Image Updater

Instructions: This tool updates enemy battler images in your enemies.json file based on matching filenames. It prioritizes matching the creature type/species (usually the second word in multi-word enemy names). The tool also checks for <Archetype: Name> tags in enemy notes. When multiple numbered variations exist (like Goblin, Goblin1, Goblin2), the tool will randomly select one of them.
